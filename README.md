# 課堂互動工具箱 MVP

這是一個可直接展示的前端 MVP，整合：

- 隨機抽籤機：匯入名單、動畫抽籤、排除已抽中者、歷史紀錄
- 小組競賽記分板：小組名稱編輯、+1 / +5 / -1、即時排序、領先組別
- 小組報告計時器：3 / 5 / 10 分鐘預設、自訂時間、倒數警示、結束提示音
- 互動測驗系統：單選、複選、是非、自動閱卷、答案解析
- 即時反饋：教師端建立單題投票、即時統計百分比

## 開發指令

```bash
npm install
npm run dev
```

## 建置指令

```bash
npm run build
```
